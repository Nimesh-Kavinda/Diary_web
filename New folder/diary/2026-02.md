## 2026-02-05

- Built the **API key rotation process**: instead of using a single Gemini API key (which hits quota limits quickly under batch generation), implemented a round-robin rotation across multiple configured API keys. Each key is tracked independently and marked as failed if it returns a quota error — the system automatically falls back to the next available key.

- **X Developer API** project day. Initialized the `x-devloper-api` project as an API exploration tool for the Twitter/X developer platform. The core problem: X API calls from the browser are blocked by CORS. Built a **Node.js backend proxy server** to receive frontend requests, sign them with the developer credentials, and forward them to the X API — completely solving the CORS restriction. Refactored the application structure to cleanly separate frontend (React) from the proxy backend (Express). Fixed a button visual/interaction bug in the API explorer UI. Added comprehensive use case documentation for all available X API v2 endpoints.
